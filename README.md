# Swipiclav - Telegram Web App

Cyber Roulette - Telegram Mini App с карточкой профиля для дейтинг-приложения

## 🚀 Возможности

- **Карточка профиля** с фото-галереей и навигацией (слайдер)
- **Интересы** - переключение тегов-кнопок (выделенные и контурные стили)
- **Действия** - кнопки "Крест" (отклонить) и "Сердце" (лайк)
- **Telegram Web App SDK** - полная интеграция с Telegram
- **Адаптивный дизайн** - работает на мобильных устройствах
- **React + TypeScript** - типизированный код
- **Tailwind CSS** - стилизация через global.css

## 🛠️ Технологии

- React 18.3.1
- TypeScript
- Vite 7.3.6
- @twa-dev/sdk 7.0.0
- Tailwind CSS (через global.css)

## 📁 Структура проекта

```
src/
├── components/           # Логические компоненты
│   ├── Header.tsx       # Верхняя панель с кнопкой "Назад"
│   ├── ProfileCard.tsx  # Карточка профиля с фото-галереей
│   ├── InterestsGrid.tsx # Сетка интересов
│   └── ActionBar.tsx    # Нижняя панель действий
├── App.tsx              # Основной компонент приложения
├── ProfilePage.tsx      # Страница профиля (основной компонент)
├── main.tsx             # Точка входа
└── vite-env.d.ts        # Типы для Vite
```

## 🚀 Запуск проекта

### Разработка

```bash
npm install
npm run dev
```

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## 📱 Интеграция с Telegram

Приложение использует Telegram Web App SDK для:
- Инициализации (`TelegramWebApp.ready()`)
- Расширения на весь экран (`TelegramWebApp.expand()`)
- Отправки данных через `TelegramWebApp.sendData()`

## 🎨 Структура страницы

### Верхняя панель
- Кнопка "Назад" (стрелка влево)
- Заголовок "Поиск пары / выбор"

### Карточка профиля
- Фото с навигацией (стрелки влево/вправо)
- Текст поверх фото: имя, возраст, локация
- Краткое описание

### Интересы
- Заголовок "Интересы"
- Сетка тегов-кнопок (IT, Технологии, Прогулки и др.)
- Выделенные и контурные стили

### Нижняя панель действий
- Мелкие текстовые подсказки: "Свайп влево", "Свайп вправо"
- Крупные круглые кнопки: "Крест" (красный) и "Сердце" (желтый)

## 🔧 Настройка

### Изменение данных профиля

В файле `src/ProfilePage.tsx` можно изменить данные профиля:

```typescript
const [profile] = useState({
  name: 'Имя',
  age: 25,
  location: 'Город',
  bio: 'Описание...',
  photos: [
    'URL_фото_1',
    'URL_фото_2',
    // ...
  ]
})
```

### Настройка интересов

```typescript
const interests = [
  { id: 'it', name: 'IT' },
  { id: 'tech', name: 'Технологии' },
  // ...
]
```
